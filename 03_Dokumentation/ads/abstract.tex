\newcommand{\deAbstractContent}{
    Im Rahmen dieser Studienarbeit wird die Entwicklung, der Bau und die Validierung eines Lidar-Systems zur Innenraumvermessung beschrieben. Das System soll sich durch einfache Handhabung und geringen Kosten von bereits auf dem Markt erhältlichen Systemen abheben. Hierzu wurde zunächst die Distanzbestimmung mittels Licht sowohl theoretisch als auch praktisch untersucht. Danach wurden diese Erkenntnisse verwendet, um geeignete Komponenten für das System auszuwählen. Sowohl die Hardware- als auch Softwareentwicklung  werden von Grund auf beschrieben. Eine ausführliche Validierung des entstandenen Systems wurde mittels verschiedener Tests zur Genauigkeit, Auflösung und Geschwindigkeit durchgeführt. Erweiterungs- und Optimierungsmöglichkeiten für das System werden erläutert.
}

\newcommand{\enAbstractContent}{
    In the context of this thesis the development, the construction and the validation of a Lidar system for interior measurement is described. The system should stand out from other systems already available on the market due to its easy handling and low costs. For this purpose, the distance determination by means of light was first examined both theoretically and practically. These findings were then used to select suitable components for the system. Both hardware and software development are described from the ground up. A detailed validation of the resulting system was carried out using various tests for accuracy, resolution and speed. Extension and optimization possibilities for the system are explained.
}
